Sastra Blog

Repository ini berisi blog sederhana untuk mempublikasikan karya sastra seperti cerpen, puisi, dan tulisan pendek.

Blog ini dibuat menggunakan HTML, CSS, dan JavaScript tanpa framework, sehingga ringan, mudah dipelajari, dan mudah dikembangkan.

Konten tulisan disimpan sebagai file HTML, sedangkan daftar posting dikelola melalui "posts.json".

Website ini dapat dijalankan secara lokal atau di-deploy menggunakan GitHub Pages.
